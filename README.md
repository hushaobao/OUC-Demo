### 功能描述

#### 实现功能

1、对输入的照片，进行几何校正（电视外框是标准长方形，但是用户拍摄出来的图片不一定端正，可能发生形变）  
2、生成图片，自动填充背景墙花纹、纹理、色彩。  
3、可实现对重复花纹、规律纹理、纯色背景的自动填充  
4、生成图片的纹理，自动与背景墙对齐  

#### 界面使用PyQT5编写(比较简洁)
